# # Classic-Clean Website

Premium Landingpage für **Classic-Clean**, einen professionellen Gebäudereinigungsdienstleister für Unternehmen in Berlin & Brandenburg.

Die Website ist als moderne, mobile-first Unternehmensseite aufgebaut und fokussiert sich auf Vertrauen, klare Kommunikation und schnelle Kontaktaufnahme.

---

## Projektübersicht

Classic-Clean präsentiert seine Reinigungsleistungen für Unternehmen, Büros, Gewerbeflächen und Einrichtungen.  
Ziel der Website ist es, Besucher schnell zu informieren und zu einer Kontaktaufnahme zu führen.

Die Startseite enthält:

- Hero-Bereich mit klarer Positionierung
- Trust- und Vorteilssektionen
- Leistungsübersicht
- Ablauf der Zusammenarbeit
- Kontaktmöglichkeiten per Telefon, WhatsApp und E-Mail
- Mobile Sticky CTA für schnelle Kontaktaufnahme

---

## Ziel der Website

Die Website soll potenziellen Kunden direkt vermitteln:

- Wer Classic-Clean ist
- Welche Reinigungsleistungen angeboten werden
- Warum Classic-Clean vertrauenswürdig ist
- Wie einfach eine Anfrage gestellt werden kann
- Dass ein Angebot schnell und unverbindlich möglich ist

---

## Designrichtung

Das Design verfolgt eine ruhige, professionelle und hochwertige Richtung:

- Cleanes Layout
- Mobile-first Struktur
- Klare Typografie
- Viel Weißraum
- Navy, Blau und Emerald als Hauptfarben
- Dezente Schatten und abgerundete Cards
- Conversion-starke Call-to-Action Buttons
- Minimalistische Animationen ohne übertriebene Effekte

Die Website soll nicht wie ein generisches Template wirken, sondern wie eine seriöse Premium-Unternehmensseite für B2B-Kunden.

## Kernfeatures

### Hero Section

Der Hero-Bereich kommuniziert direkt das Hauptangebot:

> Professionelle Gebäudereinigung für Unternehmen in Berlin & Brandenburg.

Ziel ist eine klare Conversion-Führung mit einem dominanten Hauptbutton und reduzierten sekundären Kontaktoptionen.

### TrustBar

Die TrustBar zeigt zentrale Vertrauensargumente:

- Persönlicher Ansprechpartner
- Flexible Reinigungslösungen
- Angebot in 24 Stunden
- Zuverlässige Betreuung

### Why Section

Die Why Section erklärt, warum Unternehmen mit Classic-Clean arbeiten sollten:

- Persönliche Betreuung
- Klare Kommunikation
- Flexible Einsatzplanung
- B2B-Fokus
- Kostenlose Erstberatung
- Berlin & Brandenburg

### Mobile Sticky CTA

Auf mobilen Geräten gibt es eine fixe Kontaktleiste mit:

- Anrufen
- WhatsApp
- E-Mail

Die Leiste soll dezent, conversion-stark und nicht störend wirken.

## Lokale Entwicklung

### 1. Repository klonen

```bash
git clone <REPOSITORY_URL>
cd <PROJECT_FOLDER>
```

### 2. Abhängigkeiten installieren

```bash
npm install
```

### 3. Entwicklungsserver starten

```bash
npm run dev
```

Die lokale Website ist danach üblicherweise erreichbar unter:

```
http://localhost:5173
```

## Designprinzipien

Bei zukünftigen Änderungen bitte beachten:

1. Mobile First
2. Weniger Elemente, stärkere Hierarchie
3. Eine klare Hauptaktion pro Abschnitt
4. Keine überladenen Animationen
5. Keine generischen KI-Layoutmuster
6. Premium durch Ruhe, Abstand und Klarheit
7. Conversion durch Vertrauen statt Druck
